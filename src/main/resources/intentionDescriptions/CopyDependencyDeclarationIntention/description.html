Copies a ready-to-paste Gradle Kotlin-DSL dependency declaration for the version-catalog entry under the caret to the clipboard.

On a [libraries] or [bundles] entry it copies an implementation(...) call; on a [plugins] entry it copies an alias(...) call. The catalog prefix is derived from the file name (libs.versions.toml → libs). 


# in libs.versions.toml
squareup-okhttp3 = { group = "com.squareup.okhttp3", name = "okhttp", version.ref = "squareupOkhttp3" }
copies 

implementation(libs.squareup.okhttp3)
